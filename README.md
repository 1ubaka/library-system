# Библиотечна система - Домашна работа на Любослав Илиев 22217

## Описание на проекта

Това е мини библиотечна система. Системата демонстрира основни ООП концепции като капсулация, конструктори, деструктори, статични членове, Rule of 3/5 и валидация на данни.

## Структура на класовете

### 1. Author (Автор)
- **Членове**: име, година на раждане
- **Функционалност**: съхранява информация за автори с валидация на годината (1850-2025)
- **Конструктори**: по подразбиране и параметризиран със списък за инициализация

### 2. Book (Книга)
- **Членове**: заглавие, автор, година, цена, ISBN
- **Статичен член**: `totalBooks` - брои всички живи инстанции на книги
- **Rule of 5**: имплементирани са всички 5 специални функции (копиращ/преместващ конструктор, копиращ/преместващ оператор =, деструктор)
- **Валидация**: цената не може да е отрицателна, годината трябва да е между 1000 и 2025

### 3. Member (Читател)
- **Членове**: име, ID на член, година на присъединяване
- **Валидация**: ID не може да е празен, годината трябва да е между 1900 и 2025

### 4. Loan (Заем)
- **Членове**: ISBN, ID на член, начална дата, крайна дата, статус на връщане
- **Функционалност**: 
  - Маркиране на книга като върната
  - Проверка за просрочени заеми (лексикографско сравнение на дати в YYYY-MM-DD формат)
  - Валидация: крайната дата не може да е преди началната

### 5. Library (Библиотека)
- **Членове**: вектори от книги, членове и заеми
- **Функционалност**:
  - Добавяне на книги и членове
  - Заемане и връщане на книги
  - Проверка дали книга е налична
  - Търсене на книги по автор
  - Статистика на библиотеката

## Компилация

```bash
g++ -std=c++17 -Wall -Wextra -O2 -o library main.cpp Author.cpp Book.cpp Member.cpp Loan.cpp Library.cpp
```

## Стартиране


```bash
./library
```
